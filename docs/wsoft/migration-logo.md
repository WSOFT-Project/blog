2022年4月から、WSOFTのロゴを変更します。この変更でWebSailingシリーズのロゴが代わることはありません。また、このロゴからダークモードにも対応しました。

旧バージョンのロゴも新バージョンのロゴもWSOFTダウンロードセンターからダウンロードできます。
